# RCFerreira_LoginByTaxvat module


## Descrição

Módulo usado para o customer realizar o login com o seu CPF

## Instalação

<b>composer require rcferreira/module-login-by-taxvat</b><br/>
<b>php bin/magento setup:upgrade</b>
